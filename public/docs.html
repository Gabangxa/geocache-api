GeoCache API

The geocoding API that charges you only for new lookups

GeoCache is a cache-first geocoding proxy. Every address you look up is stored in our shared cache for 24 hours (or longer on paid plans). If any user has already looked up the same address, you get the result instantly — free of charge.

This makes GeoCache dramatically cheaper for applications with repeated lookups: logistics apps, real estate platforms, delivery routing engines, and data pipelines that process addresses seen before.

Authentication

Pass your API key in the X-API-Key header on every request (except /health).

curl -X POST https://geocache-api.replit.app/v1/geocode \
  -H "Content-Type: application/json" \
  -H "X-API-Key: gca_demo_key" \
  -d '{"address": "Eiffel Tower, Paris, France"}'

How Caching Works

Every geocoding request follows this flow. Credits are only consumed on cache misses — when the address has never been seen before (or the cache has expired).

📬

Request arrives

Address string is normalized (lowercase, trimmed)

→
🗄️

Cache checked

Key looked up in shared TTL cache

→
⚡

HIT: free return

Result returned instantly. Zero credits consumed.

🌐

MISS: upstream call

1 credit consumed. Result cached for next caller.

The cache field in every response is either "hit" or "miss". The ttl_seconds field tells you how long the cached result remains valid.

POST /v1/geocode

Convert a human-readable address to latitude/longitude coordinates.

Request body

FieldTypeRequiredDescription
addressstringYesThe address string to geocode

Response fields

FieldTypeDescription
latnumberLatitude in decimal degrees
lngnumberLongitude in decimal degrees
formattedstringNormalized address string
confidenceintegerMatch confidence 1–10
cache"hit"|"miss"Whether result came from cache
ttl_secondsintegerSeconds until cache entry expires

curl example

curl -X POST https://geocache-api.replit.app/v1/geocode \
  -H "Content-Type: application/json" \
  -H "X-API-Key: gca_demo_key" \
  -d '{"address": "Eiffel Tower, Paris, France"}'

Response — cache miss (first call)

{
  "lat": 48.8584,
  "lng": 2.2945,
  "formatted": "Eiffel Tower, Champ de Mars, 5 Av. Anatole France, 75007 Paris, France",
  "confidence": 10,
  "cache": "miss",
  "ttl_seconds": 86400
}

Response — cache hit (same address again)

{
  "lat": 48.8584,
  "lng": 2.2945,
  "formatted": "Eiffel Tower, Champ de Mars, 5 Av. Anatole France, 75007 Paris, France",
  "confidence": 10,
  "cache": "hit",
  "ttl_seconds": 86389
}
